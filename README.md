# 台股全市場策略研究室

深色響應式台股分析網站，涵蓋上市、上櫃普通股與 ETF。網站提供全市場排名、AI 進場判斷、交易價格計畫、真實歷史回測、大盤濾網與投資組合風險控制。

網站內建「存股指南」，提供大盤曝險、品質門檻、分批進場、個股與 ETF 選擇，以及參考資產配置流程。

Top 10 推薦可依全部、ETF、電子、金融與傳產／其他分類切換；大盤濾網會依多頭、中性或防守狀態提供具體操作建議。

## 本機使用

在專案根目錄執行：

```powershell
python work\update_market.py
python -m http.server 8317 --directory outputs
```

接著開啟：

```text
http://127.0.0.1:8317/
```

## 策略概要

- 股票品質條件：季度 EPS 為正、年化 ROE 至少 10%、近四季營業現金流為正、負債比低於 60%、月營收年增為正。
- ETF 使用趨勢、流動性、波動與最大回撤評估，不套用公司財報。
- 大盤濾網以加權指數 60 日與 200 日均線決定建議股票曝險。
- 投組風控預設單一持股不超過資金 8%，單筆最大風險為資金 0.5%。
- 每檔標的產生「可分批進場」、「等待回測」或「暫不進場」判斷。

## 資料來源

- TWSE 與 TPEx 官方盤後、營收與財務資料
- Yahoo Finance 歷史價格與營業現金流補充資料

排名與回測不代表未來績效，也不是買賣建議。
